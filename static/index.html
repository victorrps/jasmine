Upload any PDF and get structured markdown, tables, and metadata back. Handles multi-page documents, scanned pages, and complex layouts.

Up and running in seconds

One curl command to parse your first PDF

terminal 
# Parse a PDF into structured markdown
curl -X POST http://localhost:8080/v1/parse \
  -H "X-API-Key: your_api_key" \
  -F "file=@invoice.pdf"

# Response
{
  "markdown": "# Invoice #1042\n\n| Item | Qty | Price |...",
  "page_count": 3,
  "metadata": { "title": "Invoice", ... }
}

# Extract fields with a JSON schema
curl -X POST http://localhost:8080/v1/extract \
  -H "X-API-Key: your_api_key" \
  -F "file=@invoice.pdf" \
  -F 'schema={"vendor":"string","total":"number"}'
